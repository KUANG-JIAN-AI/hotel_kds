# 🍳 スマート厨房 KDS システム

> 現代的なレストラン向けの **厨房ディスプレイシステム（Kitchen Display System）**。  
> Flask + TailwindCSS をベースに構築され、料理の進捗管理・在庫監視・補充通知をスマートに実現します。

---

## 🧭 プロジェクト概要

**スマート厨房 KDS システム** は、レストランの厨房業務をデジタル化・可視化するためのシステムです。  
リアルタイムで料理の状態を表示し、在庫量が減少した際に警告を出すことで、  
効率的なオペレーションをサポートします。

### 🔧 主な特徴
- 📋 **KDS画面**：料理の在庫残量を自動更新・表示  
- 🧾 **メニュー管理**：料理リストの閲覧・追加・編集  
- 🔐 **ログイン機能**：シンプルなアカウント認証（Flaskベース）  
- 🌙 **ダークテーマUI**：TailwindCSSを使用したモダンデザイン
